## Jevan – Meal planner web app

**Live:** [https://eatplan.app/](https://eatplan.app/) — *काय खायचं?* (What to eat?) — plan meals, manage recipes, build shopping lists, and cook with what you have. Installable as a PWA on Android and iOS.

This project uses the following stack:

- **Framework**: Next.js (App Router, TypeScript)
- **UI**: Tailwind CSS, mobile-first layout
- **Backend/API**: Next.js Route Handlers under `app/api`
- **Database**: **Supabase Postgres** (hosted) for dishes, ingredients, and relationships

To get started:

```bash
npm install
npm run dev
```

Database and deployment details are documented in `DEPLOYMENT.md`.
